# html-css-landing
Учебный одностраничный сайт , выполненный в рамках проекта по предмету HTML / CSS.

# Используемые технологии
- HTML5 (семантическая верстка)
- CSS3
- SCSS (переменные, вложенность, структура)
- Flexbox и Grid
- Адаптивная верстка (@media)
- Hover-анимации и плавные переходы

# Адаптивность
Сайт адаптирован под:
- мобильные устройства
- планшеты
- десктоп
