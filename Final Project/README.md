# HTML / CSS Landing Page
Учебный одностраничный сайт (лендинг), выполненный в рамках проекта по предмету HTML / CSS.

## 🔧 Используемые технологии
- HTML (семантическая верстка)
- CSS
- SCSS (переменные, вложенность, структура)
- Flexbox и Grid
- Адаптивная верстка (@media)
- Hover-анимации и плавные переходы

## 📱 Адаптивность
Сайт адаптирован под:
- мобильные устройства
- планшеты
- десктоп
